# uber_clone

A **frontend web clone of the Uber UI** built using HTML, CSS, JavaScript, and PHP.  
This project replicates the look and feel of the Uber landing page / booking interface.

## Features

- Responsive landing page UI
- Ride booking form
- Static map / search UI
- Styled with CSS
- Basic routing with PHP

---

## Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript**
- **PHP**

---

## Installation

1. Clone the repo:
   git clone https://github.com/parthgarg0/uber_clone.git
   cd uber_clone
2. Run on a local PHP server:
   php -S localhost:8000
3. Open in browser:
   http://localhost:8000

---

## Usage

This project provides a frontend interface only.
There is no backend ride booking system or database implemented in this repo.

To extend it into a real Uber clone, you would need:
- A backend API (Node/PHP/Python)
- Database (MySQL / MongoDB)
- Maps and Geolocation API
- Authentication
